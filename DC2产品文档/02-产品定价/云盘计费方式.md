##云盘计费方式
云盘支持包年包月和按时长两种计费方式。

|类型         | 包年包月价格 |按时长价格  | 
| --------    | -----: | -----: |
| SSD云盘    | 0.48 元/GB/月 |   0.0008元/GB/小时  |  
| 高效云盘    | 0.18 元/GB/月  |  0.0003元/GB/小时  |  

##快照计费方式
快照不以单独资源计费，以账户下所有快照的空间使用量计费。

|快照总大小         | 收费价格 |
| --------    | -----: | 
| <=20G    | 不收费 | 
| >20G    | 0.12(x - 20) 元/GB/月  |
